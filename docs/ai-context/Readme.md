AI Context

This folder contains contextual information intended to help AI assistants understand the project when they are asked to help with development tasks.

## Purpose

- Provide high-level project description and architecture notes for AI tools.
- Supply instructions, policies, and data sources AI assistants should use or respect.

## Guidelines for AI Assistants

- Use the information in this folder only to inform suggestions and code changes.
- Do not modify files in this folder unless explicitly authorized by a human maintainer.
- When producing code changes, cite any assumptions you made that are not explicit in the docs.

## What to Put Here

- Project overview and architecture summaries
- Key design decisions and rationale
- Data schemas and API contracts
- Coding standards and linting/formatting rules
- Important repository paths and deployment notes

## Maintainer Notes

- Keep this folder concise and focused on machine-consumable context.
- When updating architecture or APIs, update these notes so AI assistants receive accurate context.
