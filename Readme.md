
# KuchTohProduct

A mindspace-like platform for managing notes and context across various file types (images, videos, PDFs, links) with cloud support, AI-powered queries, and automatic data classification.

## Features

- Organize and maintain notes and context for multiple file types
- Cloud storage and synchronization
- AI-powered search and data classification
- Support for images, videos, PDFs, and web links

## Tech Stack

- **Flutter** (Frontend)
- **Python** (Backend)
- **Supabase** (Database & Auth)

## Scalability & Infrastructure

- Kubernetes for orchestration
- Load balancing for high availability
- Designed for scalable, cloud-native deployments
